Осевой скоринг — реальная формула
S(a, b) = Σ wᵢ · score(axisᵢ(a), axisᵢ(b)) Σ wᵢ
Сравнение артикулов по физическим осям:
сечение, материал, ГОСТ, класс. Вес оси wᵢ ∈ {critical, high, medium, low}
Правила сравнения осей
exact:  1.0 | 0.0
upgrade_ok:  1.0 − 0.1·d
substitute_ok:  1.0 − 0.15·|d|
d — расстояние в цепочке апгрейда.
При critical failure — кандидат отклоняется.

Визуализация эвристического матчинга

Концептуальная модель: множество объектов слева → оценка расстояний → нахождение ближайших пар справа. Красные — объекты без пары (гэпы).

Осевой скоринг
// матрица совпадений осей
сечение 2.5 = 2.5 → 1.0 critical
класс  LS ≠ FR  → 0.0 critical
жилы   3 = 3   → 1.0 high

// critical fail → ОТКЛОНЁН

Каждая ось имеет вес и правило. Провал по критической оси = жёсткий отказ.
